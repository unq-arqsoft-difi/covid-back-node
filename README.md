## TL;DR

```sh
git clone https://github.com/unq-arqsoft-difi/covid-back-node.git
cd covid-back-node
npm i
npx sequelize db:migrate
npm run start
```

La API quedará levantada en `http://localhost:9001`

## Install & Run

### Download

```sh
git clone https://github.com/unq-arqsoft-difi/covid-back-node.git
cd covid-back-node
```

### Install

```sh
npm i
```

### Run Migrations

```sh
npx sequelize db:migrate
```

### Start

```sh
npm run start
```

La API quedará levantada en `http://localhost:9001`
